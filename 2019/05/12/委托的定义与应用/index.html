委托的定义与应用

委托的定义

delegate声明的语法：delegate int MyDel(int i,string s);
MyDel是一个数据类型

可以声明一个变量指向和他类型相容的方法

MyDel d1=new MyDel(M1);//声明MyDel类型的变量，指向一个指向M1方法的对象
//可简化为 MyDel d1=M1;
不要声明成MyDel d1=M1();
因为加上括号就代表调用方法了，代表“执行指向的方法”，参数就要按照方法的规则传递。
委托和null都是引用类型，是可以为null,如果委托变量是null，那么调用的时候就会抛出NullReferenceException

泛型委托

内置泛型委托

Action 有参数无返回值的泛型委托
